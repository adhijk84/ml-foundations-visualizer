ML Foundations Visualizer

An interactive web application to visualize core Machine Learning and Mathematical concepts such as vector similarity, derivatives, and gradient descent.

Built using Streamlit, NumPy, SymPy, Matplotlib, and Plotly, this project helps students and developers understand ML foundations visually.


Features

Vector Similarity Visualizer

* Input or generate vectors
* Compute:

  * Dot Product
  * Cosine Similarity
  * Euclidean Distance
* Visualize vectors in 2D space
* Understand angle relationships between vectors


Function & Derivative Visualizer

* Input custom mathematical functions
* Automatically compute derivatives using SymPy
* Plot:

  * Function curve
  * Derivative curve
* Helps in understanding slope and rate of change


Gradient Descent Simulator

* Simulate optimization using Gradient Descent
* Adjustable:

  * Learning rate
  * Number of iterations
  * Starting point
* Visualize convergence process
* Plot parameter updates over time

Tech Stack

 Installation & Setup

1. Clone the repository

git clone https://github.com/YOUR_USERNAME/ml-foundations-visualizer.git
cd ml-foundations-visualizer


2. Install dependencies
pip install -r requirements.txt

 3. Run the application

streamlit run app.py

 Usage

* Open the app in your browser
* Use the sidebar to switch between modules:

  * Vector Similarity
  * Function & Derivative
  * Gradient Descent
* Interact with inputs and visualize results instantly
